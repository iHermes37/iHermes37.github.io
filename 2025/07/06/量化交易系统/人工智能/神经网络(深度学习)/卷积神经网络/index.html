卷积神经网络

多层感知机十分适合处理表格数据，其中行对应样本，列对应特征。 对于表格数据，我们寻找的模式可能涉及特征之间的交互，但是我们不能预先假设任何与特征交互相关的先验结构。 此时，多层感知机可能是最好的选择，然而对于高维感知数据，这种缺少结构的网络可能会变得不实用。

引入

全连接网络在图像任务中的问题

以 224×224×3 的彩色图片为例：

  • 输入层节点数 = 224 × 224 × 3 = 150,528

  • 若第一个隐藏层有 1000 个神经元
    → 参数量 = 150,528 × 1000 = 1.5 亿参数！
    → 存储 + 计算都极其昂贵。

    参数数量太多 考虑一个输入1000 * 1000像素的图片(一百万像素，现在已经不能算大图了)，输入层有1000 * 1000=100万节点。假设第一个隐藏层有100个节点(这个数量并不多)，那么仅这一层就有(1000 *1000+1) *100=1亿参数，这实在是太多了！我们看到图像只扩大一点，参数数量就会多很多，因此它的扩展性很差。

更糟糕的是：

  • 网络对像素位置非常敏感：
    同样的物体只要稍微平移，神经元激活模式就完全不同。
  • 没有利用“局部特征”或“空间邻近性”：
    图像的结构信息（如边缘、纹理）被完全忽略。对于图像识别任务来说，每个像素和其周围像素的联系是比较紧密的，和离得很远的像素的联系可能就很小了。如果一个神经元和上一层所有神经元相连，那么就相当于对于一个像素来说，把图像的所有像素都等同看待，这不符合前面的假设。当我们完成每个连接权重的学习之后，最终可能会发现，有大量的权重，它们的值都是很小的(也就是这些连接其实无关紧要)。努力学习大量并不重要的权重，这样的学习必将是非常低效的。
  • 网络层数限制，网络层数越多其表达能力越强，但是通过梯度下降方法训练深度全连接神经网络很困难，因为全连接神经网络的梯度很难传递超过3层。因此，我们不可能得到一个很深的全连接神经网络，也就限制了它的能力。

CNN— 解决了哪些问题

局部连接

CNN 不再让每个神经元连接所有输入，而是只连接局部区域（局部感受野）。即每个神经元不再和上一层的所有神经元相连，而只和一小部分神经元相连。这样就减少了很多参数。

例如，一个 3×3 卷积核只看邻域 9 个像素。

卷积核中的数值就是神经网络的权重，也就是要学习的参数，将权重与对应输入位置的值（如果是输入层就是像素值，如果是中间层就是中间层神经元的激活值）相乘，再与偏置相加，经过激活函数，便得到了对应的输出。可以看出，其实本质上跟上面介绍的神经元的计算方式相类似，也是求加权和。

利用图像局部相关性减少参数量：大幅度降低计算复杂度

权值共享

在同一层的不同位置使用同一个卷积核（同一组权重）扫描整张图片。组连接可以共享同一个权重，而不是每个连接有一个不同的权重，这样又减少了很多参数。

相同的特征检测器（如边缘检测）可应用于整张图进一步减少参数数量（数万倍减少并提高泛化能力

可以看到相比于传统的MLP，不同之处在于：

  • 局部连接。卷积操作中相邻层的神经元是局部连接的，下一层神经元的值只取决于卷积核覆盖的窗口上的值；
  • 权重共享。在一个卷积层中，使用同一个卷积核对不同区域的数据进行处理。

平移不变性

池化层（Pooling）在卷积层后引入一种统计降采样操作，使网络对小范围的位移、变形具有鲁棒性。同一物体出现在不同位置时，网络仍能识别

典型组成结构

卷积神经网络（CNN）的卷积层和池化层是其核心组件，共同实现特征提取与降维的功能。

卷积层

卷积层通过局部感受野和权值共享机制，使用多个可学习的卷积核（滤波器）对输入数据进行扫描运算：每个卷积核在输入图像或特征图上以固定步长（stride）滑动，计算局部区域与卷积核的逐元素乘积之和并加上偏置，生成对应的特征图（Feature Map），这一过程能够有效捕捉图像中的局部特征（如边缘、纹理等）

激活层

并通过非线性激活函数（如ReLU）引入非线性变换。卷积操作通过填充（padding）控制输出尺寸，若设置padding='same'可保持空间分辨率不变，而多通道卷积则通过堆叠不同卷积核实现特征的多维度表达。

最近几年卷积神经网络中，激活函数往往不选择sigmoid或tanh函数，而是选择relu函数。Relu函数的定义是：

ReLU（Rectified Linear Unit，线性整流单元）函数的定义如下：
$$
f(x) = \max(0, x)
$$
也就是说：
$$
f(x) = \begin{cases} x, & x > 0 \ 0, & x \le 0 \end{cases}
$$
ReLU 就像一个“单向开关”：

  • 输入为正数时，直接通过（线性部分）
  • 输入为负数时，全部截断为 0

它让神经元在“激活”和“抑制”之间切换，产生稀疏激活（Sparse Activation），即只有部分神经元真正起作用。

Relu函数作为激活函数，有下面几大优势：

  • 速度快：和sigmoid函数需要计算指数和倒数相比，relu函数其实就是一个max(0,x)，计算代价小很多。

  • 减轻梯度消失问题：回忆一下计算梯度的公式。其中，是sigmoid函数的导数。在使用反向传播算法进行梯度计算时，每经过一层sigmoid神经元，梯度就要乘上一个。从下图可以看出，函数最大值是1/4。因此，乘一个会导致梯度越来越小，这对于深层网络的训练是个很大的问题。而relu函数的导数是1，不会导致梯度变小。当然，激活函数仅仅是导致梯度减小的一个因素，但无论如何在这方面relu的表现强于sigmoid。使用relu激活函数可以让你训练更深的网络。

  • 稀疏性 通过对大脑的研究发现，大脑在工作的时候只有大约5%的神经元是激活的，而采用sigmoid激活函数的人工神经网络，其激活率大约是50%。有论文声称人工神经网络在15%-30%的激活率时是比较理想的。因为relu函数在输入小于0时是完全不激活的，因此可以获得一个更低的激活率。

池化层

池化层则紧随卷积层之后，主要作用是通过降采样（如最大池化选取局部区域最大值或平均池化计算均值）减少参数量和计算复杂度，同时增强模型对微小位移的鲁棒性（平移不变性），

最大池化（Max Pooling）通过保留局部最显著特征来突出关键信息，而平均池化（Average Pooling）则提供平滑的降采样效果。两者交替堆叠形成层级结构，

低层卷积捕捉基础视觉特征（如线条、色块），高层卷积组合低层特征形成更复杂的语义特征（如物体部件），

池化层逐步压缩空间维度，最终通过展平操作（Flatten）连接全连接层完成分类任务。

这种结构显著降低了参数数量（相比全连接网络），保留了空间信息，并赋予CNN对平移、缩放和旋转的固有不变性，使其成为图像处理任务的经典架构。

全连接层

全连接层（Fully Connected Layer）是神经网络中的核心组件之一，通常位于卷积神经网络（CNN）的末端，负责将经过卷积和池化操作提取的高维特征映射转换为最终的分类或回归结果。

特点是层内所有神经元与前一层的每个输出完全连接，形成稠密的权重矩阵，能够综合全局信息进行决策。在CNN中，全连接层通常接收由前面卷积和池化层生成并展平后的特征向量（例如将三维的32×7×7特征图展平为1568维向量），通过线性变换（即矩阵乘法加偏置）和非线性激活函数（如ReLU或Softmax）的配合，实现从抽象特征到具体类别概率分布的映射。通过权重矩阵实现特征线性组合。其数学本质是矩阵乘法运算：y = Wx + b。

以分类任务为例，最后一个全连接层的输出维度通常等于类别数量（如MNIST数字识别为10维），配合Softmax激活函数输出每个类别的概率。全连接层的参数量较大（由输入和输出维度乘积决定），容易导致过拟合，因此常配合Dropout或正则化技术使用。虽然近年来全局平均池化（Global Average Pooling）等结构逐渐替代了部分全连接层以减少参数，但全连接层在特征整合和最终决策中仍扮演着不可替代的角色，尤其在需要综合全局信息的任务中表现突出。

运行过程

一个典型 CNN（如 ResNet、VGG、LeNet）的训练过程如下：

  1. 前向传播（Forward Propagation）

    输入图片 → 卷积层提取特征 → 激活层非线性变换 → 池化层降采样 → 全连接层 → 输出分类概率。输出为预测值 。

  2. 计算损失（Loss Calculation）

    比较预测值与真实标签，计算损失函数。

  3. 反向传播（Backpropagation）

    利用链式法则计算每个参数（卷积核、权重、偏置）的梯度。

  4. 参数更新（Parameter Update）

    使用优化算法（如 SGD、Adam）根据梯度更新参数，逐步降低损失。

  5. 重复迭代（Iteration / Epoch）

    多轮训练，直到损失收敛或验证集性能最优。

损失函数

损失函数的选择取决于任务类型：

反向传播

与普通全连接网络不同，CNN 的反向传播要考虑卷积运算特性。

  1. 卷积层梯度计算

    对卷积核权重的梯度：
    $$
    \frac{\partial L}{\partial W} = x * \frac{\partial L}{\partial y}
    $$
    （这里 * 表示卷积操作），对输入的梯度：

    $$
    \frac{\partial L}{\partial x} = \frac{\partial L}{\partial y} * W_{rot180}
    $$

    （卷积核翻转 180°）

  2. 池化层梯度传播

    • Max Pooling：仅将梯度传给最大值所在位置；
    • Average Pooling：梯度均匀分配给窗口内所有位置。

  3. 激活函数梯度

    ReLU：
    $$
    \frac{df}{dx} =
    \begin{cases}
    1, & x>0
    0, & x\le 0
    \end{cases}
    $$

优化算法

优化算法 特点
SGD（随机梯度下降） 基础方法，稳定但收敛慢
SGD + Momentum 加速收敛、减少震荡
Adam 自适应学习率，收敛快
RMSProp 适用于非平稳目标（如RNN）
AdamW 引入权重衰减，防止过拟合（常用于ResNet、Transformer）

更新规则以 Adam 为例：
$$
\theta_{t+1} = \theta_t - \eta \frac{\hat{m}_t}{\sqrt{\hat{v}_t} + \epsilon}
$$

正则化与提升技巧

技术 作用
Dropout 随机丢弃神经元，防止过拟合
Batch Normalization (BN) 稳定梯度、加速训练
Data Augmentation 扩充数据（翻转、旋转、裁剪）提高泛化性
Weight Decay (L2正则化) 限制权重大小，避免过拟合
Learning Rate Schedule 逐步降低学习率以更精细地收敛
