  • зҠ¶жҖҒпјҲState, sпјүпјҡзҺҜеўғеңЁжҹҗдёҖж—¶еҲ»зҡ„жҸҸиҝ°гҖӮ

  • еҠЁдҪңпјҲAction, aпјүпјҡжҷәиғҪдҪ“еңЁз»ҷе®ҡзҠ¶жҖҒдёӢеҸҜжү§иЎҢзҡ„ж“ҚдҪңгҖӮ

  • зӯ–з•ҘпјҲPolicy, ПҖпјүпјҡжҷәиғҪдҪ“зҡ„иЎҢдёәи§„еҲҷпјҢеҚід»ҺзҠ¶жҖҒеҲ°еҠЁдҪңзҡ„жҳ е°„пјҲзЎ®е®ҡжҖ§жҲ–йҡҸжңәжҖ§пјүгҖӮ

  • д»·еҖјеҮҪж•°пјҲValue FunctionпјүпјҡиҜ„дј°зҠ¶жҖҒзҡ„й•ҝжңҹд»·еҖјпјҢеҲҶдёәпјҡ

    • **зҠ¶жҖҒд»·еҖјеҮҪж•° VПҖ(s)**пјҡеңЁзӯ–з•Ҙ ПҖ дёӢд»ҺзҠ¶жҖҒ sејҖе§Ӣзҡ„жңҹжңӣзҙҜз§ҜеҘ–еҠұгҖӮ

  • зҺҜеўғжЁЎеһӢпјҲModelпјүпјҲеҸҜйҖүпјүпјҡеҜ№зҺҜеўғеҠЁжҖҒпјҲзҠ¶жҖҒиҪ¬з§»е’ҢеҘ–еҠұпјүзҡ„йў„жөӢгҖӮ

ејәеҢ–еӯҰд№ зҡ„зӣ®ж ҮжҳҜжүҫеҲ°жңҖдјҳзӯ–з•Ҙ ПҖвҲ—пјҢжңҖеӨ§еҢ–зҙҜз§ҜеҘ–еҠұгҖӮж ёеҝғй—®йўҳеҢ…жӢ¬пјҡ

  • жҺўзҙў vs еҲ©з”Ёпјҡе№іиЎЎе°қиҜ•ж–°еҠЁдҪңпјҲиҺ·еҸ–дҝЎжҒҜпјүдёҺйҖүжӢ©е·ІзҹҘжңҖдјҳеҠЁдҪңпјҲжңҖеӨ§еҢ–еҘ–еҠұпјүгҖӮ
  • дҝЎз”ЁеҲҶй…Қпјҡе°Ҷй•ҝжңҹз»“жһңеҪ’еӣ дәҺзҹӯжңҹеҠЁдҪңгҖӮ

еҪ“дҪҝз”Ёж·ұеәҰзҘһз»ҸзҪ‘з»ңпјҲDNNпјүжқҘиҝ‘дјјQеҮҪж•°ж—¶пјҢдј з»ҹзҡ„зӣ‘зқЈеӯҰд№ ж–№жі•дёҚиғҪзӣҙжҺҘеә”з”ЁпјҢеӣ дёәзјәд№ҸвҖңзңҹе®һзҡ„QеҖјвҖқдҪңдёәж ҮзӯҫгҖӮдҪҶйҖҡиҝҮејәеҢ–еӯҰд№ зҡ„ж—¶еәҸе·®еҲҶпјҲжҖқжғіпјҢеҸҜд»Ҙжһ„йҖ дёҖдёӘвҖңиҮӘжҙҪвҖқзҡ„и®ӯз»ғзӣ®ж ҮгҖӮ)

з»ҸйӘҢеӣһж”ҫ

еңЁдёҖиҲ¬зҡ„жңүзӣ‘зқЈеӯҰд№ дёӯпјҢеҒҮи®ҫи®ӯз»ғж•°жҚ®жҳҜзӢ¬з«ӢеҗҢеҲҶеёғзҡ„пјҢжҲ‘д»¬жҜҸж¬Ўи®ӯз»ғзҘһз»ҸзҪ‘з»ңзҡ„ж—¶еҖҷд»Һи®ӯз»ғж•°жҚ®дёӯйҡҸжңәйҮҮж ·дёҖдёӘжҲ–иӢҘе№ІдёӘж•°жҚ®жқҘиҝӣиЎҢжўҜеәҰдёӢйҷҚпјҢйҡҸзқҖеӯҰд№ зҡ„дёҚж–ӯиҝӣиЎҢпјҢжҜҸдёҖдёӘи®ӯз»ғж•°жҚ®дјҡиў«дҪҝз”ЁеӨҡж¬ЎгҖӮеңЁеҺҹжқҘзҡ„ Q-learning з®—жі•дёӯпјҢжҜҸдёҖдёӘж•°жҚ®еҸӘдјҡз”ЁжқҘжӣҙж–°дёҖж¬ЎеҖјгҖӮдёәдәҶжӣҙеҘҪең°е°Ҷ Q-learning е’Ңж·ұеәҰзҘһз»ҸзҪ‘з»ңз»“еҗҲпјҢDQN з®—жі•йҮҮз”ЁдәҶз»ҸйӘҢеӣһж”ҫпјҲexperience replayпјүж–№жі•пјҢе…·дҪ“еҒҡжі•дёәз»ҙжҠӨдёҖдёӘеӣһж”ҫзј“еҶІеҢәпјҢе°ҶжҜҸж¬Ўд»ҺзҺҜеўғдёӯйҮҮж ·еҫ—еҲ°зҡ„еӣӣе…ғз»„ж•°жҚ®пјҲзҠ¶жҖҒгҖҒеҠЁдҪңгҖҒеҘ–еҠұгҖҒдёӢдёҖзҠ¶жҖҒпјүеӯҳеӮЁеҲ°еӣһж”ҫзј“еҶІеҢәдёӯпјҢи®ӯз»ғ Q зҪ‘з»ңзҡ„ж—¶еҖҷеҶҚд»Һеӣһж”ҫзј“еҶІеҢәдёӯйҡҸжңәйҮҮж ·иӢҘе№Іж•°жҚ®жқҘиҝӣиЎҢи®ӯз»ғгҖӮиҝҷд№ҲеҒҡеҸҜд»Ҙиө·еҲ°д»ҘдёӢдёӨдёӘдҪңз”ЁгҖӮ

зҺ°еңЁзҡ„й—®йўҳе°ұе·Із»ҸиҪ¬жҚўдёәйңҖиҰҒдёҖз»„и®ӯз»ғйӣҶпјҢе®ғиғҪеӨҹжҸҗдҫӣдёҖжү№еӣӣе…ғз»„пјҲs, a, r, sвҖҷпјүпјҢе…¶дёӯsвҖҷдёәsжү§иЎҢaеҗҺзҡ„дёӢдёҖдёӘзҠ¶жҖҒгҖӮеҰӮжһңиғҪжңүиҝҷж ·дёҖдёӘеӣӣе…ғз»„пјҢе°ұиғҪеӨҹз”ЁжқҘи®ӯз»ғDNNдәҶпјҢиҝҷе°ұжҳҜжҲ‘д»¬иҰҒд»Ӣз»Қзҡ„з»ҸйӘҢеӣһж”ҫзј“еӯҳгҖӮ

дёәд»Җд№ҲйңҖиҰҒз»ҸйӘҢеӣһж”ҫпјҹ

  • жү“з ҙж•°жҚ®зӣёе…іжҖ§пјҡзӣҙжҺҘжҢүж—¶й—ҙйЎәеәҸи®ӯз»ғпјҲеҰӮжңҖж–°ж ·жң¬жҖ»жҳҜдёҖиҝһдёІвҖңи·іи·ғвҖқеҠЁдҪңпјүдјҡеҜјиҮҙDNNвҖңеҒҸз§‘вҖқпјҢеҸӘж“…й•ҝеӨ„зҗҶиҝ‘жңҹзҠ¶жҖҒгҖӮйҡҸжңәжҠҪеҸ–ж—§ж ·жң¬и®ӯз»ғпјҢиғҪи®©жЁЎеһӢеӯҰеҲ°жӣҙе…Ёйқўзҡ„зӯ–з•ҘгҖӮ
    зұ»жҜ”пјҡиҖғиҜ•еүҚдёҚиғҪеҸӘеӨҚд№ жңҖеҗҺдёҖз« пјҢиҖҢиҰҒд»Һж•ҙдёӘиҜҫжң¬дёӯйҡҸжңәжҠҪйўҳз»ғд№ гҖӮ
  • йҮҚеӨҚеҲ©з”Ёж•°жҚ®пјҡжҜҸдёӘж ·жң¬еҸҜд»Ҙиў«еӨҡж¬ЎдҪҝз”ЁпјҲиҠӮзңҒдёҺзҺҜеўғдәӨдә’зҡ„жҲҗжң¬пјүгҖӮ

жғіиұЎеңЁзҺ©дёҖдёӘй—Ҝе…іжёёжҲҸпјҡжҜҸжү§иЎҢдёҖдёӘеҠЁдҪңпјҲжҜ”еҰӮвҖңи·іи·ғвҖқпјүпјҢдјҡи®°дҪҸпјҡ

  • еҪ“еүҚзҠ¶жҖҒпјҲsпјҡзңӢеҲ°йҡңзўҚзү©пјү
  • жү§иЎҢзҡ„еҠЁдҪңпјҲaпјҡи·іи·ғпјү
  • еҫ—еҲ°зҡ„еҘ–еҠұпјҲrпјҡ+10еҲҶпјҢеӣ дёәи·іиҝҮйҡңзўҚпјү
  • дёӢдёҖдёӘзҠ¶жҖҒпјҲsвҖҷпјҡиҗҪең°еҗҺзҡ„ж–°з”»йқўпјү
  • жҳҜеҗҰз»“жқҹпјҲdoneпјҡFalseпјҢеӣ дёәиҝҳжІЎйҖҡе…іпјү

иҝҷдәӣдҝЎжҒҜжһ„жҲҗдёҖжқЎиҪ¬з§»ж ·жң¬пјҲs, a, r, sвҖҷ, doneпјүпјҢеӯҳе…ҘдёҖдёӘвҖңи®°еҝҶеә“вҖқпјҲеҚіReplay BufferпјүгҖӮ

еүҚйқўжҸҗеҲ°жҲ‘д»¬йңҖиҰҒдёҖжү№еӣӣе…ғз»„пјҲs, a, r, sвҖҷпјүжқҘиҝӣиЎҢи®ӯз»ғпјҢеӣ жӯӨжҲ‘д»¬йңҖиҰҒзј“еӯҳдёҖжү№иҝҷж ·зҡ„еӣӣе…ғз»„еҲ°з»ҸйӘҢжұ дёӯд»Ҙдҫӣи®ӯз»ғд№Ӣз”ЁгҖӮз”ұдәҺжҜҸж¬Ўжү§иЎҢдёҖдёӘеҠЁдҪңеҗҺйғҪиғҪиҪ¬з§»еҲ°дёӢдёҖдёӘзҠ¶жҖҒпјҢе№¶иҺ·еҫ—дёҖдёӘrewardпјҢеӣ жӯӨжҲ‘д»¬жҜҸжү§иЎҢдёҖж¬ЎеҠЁдҪңеҗҺйғҪеҸҜд»ҘиҺ·еҫ—дёҖдёӘиҝҷж ·зҡ„еӣӣе…ғз»„пјҢд№ҹеҸҜд»Ҙе°ҶиҝҷдёӘеӣӣе…ғз»„зӣҙжҺҘж”ҫе…Ҙз»ҸйӘҢжұ дёӯгҖӮ

и®ӯз»ғж ·жң¬зҡ„жқҘжәҗ

  • з»ҸйӘҢеӣһж”ҫзј“еӯҳпјҲReplay BufferпјүпјҡеӯҳеӮЁжҷәиғҪдҪ“дёҺзҺҜеўғдәӨдә’зҡ„иҪ¬з§»ж ·жң¬ (s,a,r,sвҖІ,done)е…¶дёӯ doneиЎЁзӨәжҳҜеҗҰз»ҲжӯўзҠ¶жҖҒпјҲеҰӮжёёжҲҸз»“жқҹпјүгҖӮ
  • йҮҮж ·ж–№ејҸпјҡд»Һзј“еӯҳдёӯйҡҸжңәжҠҪеҸ–дёҖжү№ж ·жң¬пјҲжү“з ҙж•°жҚ®зӣёе…іжҖ§пјҢжҸҗеҚҮзЁіе®ҡжҖ§пјүгҖӮ

жҲ‘д»¬зҹҘйҒ“иҝҷз§Қеӣӣе…ғз»„д№Ӣй—ҙжҳҜеӯҳеңЁе…іиҒ”жҖ§зҡ„пјҢеӣ дёәзҠ¶жҖҒзҡ„иҪ¬з§»жҳҜиҝһз»ӯзҡ„пјҢеҰӮжһңзӣҙжҺҘжҢүйЎәеәҸеҸ–дёҖжү№еӣӣе…ғз»„дҪңдёәи®ӯз»ғйӣҶпјҢйӮЈд№ҲжҳҜе®№жҳ“иҝҮжӢҹеҗҲзҡ„пјҢеӣ дёәи®ӯз»ғж ·жң¬й—ҙдёҚжҳҜзӢ¬з«Ӣзҡ„пјҒдёәи§ЈеҶіиҝҷдёӘй—®йўҳпјҢжҲ‘д»¬еҸҜд»Ҙз®ҖеҚ•ең°д»Һз»ҸйӘҢжұ дёӯйҡҸжңәжҠҪеҸ–е°‘йҮҸеӣӣе…ғз»„дҪңдёәдёҖдёӘbatchпјҢиҝҷж ·ж—ўдҝқиҜҒдәҶи®ӯз»ғж ·жң¬жҳҜзӢ¬з«ӢеҗҢеҲҶеёғзҡ„пјҢд№ҹдҪҝеҫ—жҜҸдёӘbatchж ·жң¬йҮҸдёҚеӨ§пјҢиғҪеҠ еҝ«и®ӯз»ғйҖҹеәҰгҖӮ

зӣ®ж ҮзҪ‘з»ң

зҺ°еңЁжҲ‘д»¬зҡ„йҖүжӢ©е“ӘдёӘеҠЁдҪңпјҢжҳҜз”ұDNNжқҘеҒҡеҶіе®ҡзҡ„пјҢеӣ жӯӨжҲ‘д»¬йңҖиҰҒи®ӯз»ғDNNд»ҘдҪҝе…¶иғҪиҫҫеҲ°д»Өдәәж»Ўж„Ҹзҡ„иЎЁзҺ°гҖӮиҝҷжҳҫз„¶жҳҜдёҖдёӘзӣ‘зқЈеӯҰд№ зҡ„й—®йўҳпјҢйӮЈд№Ҳи®ӯз»ғйӣҶжҳҜд»Җд№ҲпјҢж ҮзӯҫжҳҜд»Җд№ҲпјҢжҚҹеӨұеҮҪж•°еҸҲжҳҜд»Җд№Ҳпјҹ

йҰ–е…ҲпјҢжҲ‘д»¬DNNзҡ„иҫ“еҮәеҖјпјҢиҮӘз„¶жҳҜеңЁз»ҷе®ҡзҠ¶жҖҒзҡ„жғ…еҶөдёӢпјҢжү§иЎҢеҗ„actionеҗҺиғҪеҫ—еҲ°зҡ„QеҖјгҖӮз„¶иҖҢдәӢе®һдёҠжҲ‘д»¬еңЁеҫҲеӨҡжғ…еҶөдёӢе№¶дёҚзҹҘйҒ“жңҖдјҳзҡ„QеҖјжҳҜд»Җд№ҲпјҢжҜ”еҰӮиҮӘеҠЁй©ҫй©¶гҖҒеӣҙжЈӢзӯүжғ…еҶөпјҢжүҖд»Ҙдјјд№ҺжҲ‘д»¬жІЎжі•з»ҷеҮәж ҮзӯҫгҖӮдҪҶжҳҜд»Җд№ҲжҳҜдёҚеҸҳзҡ„е‘ўпјҹRewardпјҒ

еҰӮдҪ•йҖҡиҝҮRewardи®ӯз»ғQзҪ‘з»ңпјҹ

ж ёеҝғй—®йўҳ:

  • жІЎжңүзңҹе®һQеҖјж ҮзӯҫпјҡеңЁиҮӘеҠЁй©ҫй©¶жҲ–еӣҙжЈӢдёӯпјҢжҲ‘д»¬ж— жі•йў„е…ҲзҹҘйҒ“вҖңеңЁжҹҗдёӘзҠ¶жҖҒдёӢе·ҰиҪ¬зҡ„зІҫзЎ®QеҖјеә”иҜҘжҳҜеӨҡе°‘вҖқгҖӮ
  • е”ҜдёҖе·ІзҹҘпјҡжү§иЎҢеҠЁдҪңеҗҺзҡ„еҚіж—¶еҘ–еҠұrпјҲеҰӮвҖңж’һиҪҰпјҡ-1000вҖқпјҢвҖңе®үе…ЁиЎҢй©¶пјҡ+1вҖқпјүгҖӮ

и§ЈеҶіж–№жЎҲпјҡиҙқе°”жӣјж–№зЁӢ

йҖҡиҝҮд»ҘдёӢе…¬ејҸйҖ’еҪ’е®ҡд№үQеҖјпјҡ$Q(s,a)=r+ОіmaxвҒЎaвҖІQ(sвҖІ,aвҖІ)$ еҚіпјҡеҪ“еүҚQеҖј = еҚіж—¶еҘ–еҠұ + жңӘжқҘеҸҜиғҪзҡ„жңҖеӨ§ж”¶зӣҠзҡ„жҠҳжүЈ

иҝҷж ·пјҢеҚідҪҝдёҚзҹҘйҒ“зңҹе®һзҡ„ QвҲ—пјҢд№ҹеҸҜд»ҘйҖҡиҝҮдёҚж–ӯиҝӯд»ЈйҖјиҝ‘е®ғгҖӮ

Target (r+ОіmaxвҒЎQ)дёӯпјҢrжҳҜзңҹе®һи§ӮеҜҹеҲ°зҡ„еҘ–еҠұпјҢиҖҢ maxвҒЎQ жҳҜDNNеҜ№жңӘжқҘеҘ–еҠұзҡ„йў„жөӢгҖӮйҖҡиҝҮжңҖе°ҸеҢ–TDиҜҜе·®пјҢDNNе®һйҷ…дёҠжҳҜеңЁеӯҰд№ пјҡеҰӮдҪ•е°ҶеҪ“еүҚзҡ„еҚіж—¶еҘ–еҠұ rе’ҢжңӘжқҘйў„жөӢзҡ„еҘ–еҠұ ОіmaxвҒЎQз»„еҗҲиө·жқҘпјҢеҪўжҲҗеҜ№еҠЁдҪңжҖ»д»·еҖјзҡ„еҮҶзЎ®дј°и®ЎгҖӮ

еҜ№зҠ¶жҖҒsпјҢжү§иЎҢеҠЁдҪңaпјҢйӮЈд№Ҳеҫ—еҲ°зҡ„rewardжҳҜдёҖе®ҡзҡ„пјҢиҖҢдё”жҳҜдёҚеҸҳзҡ„пјҒеӣ жӯӨйңҖиҰҒиҖғиҷ‘д»ҺrewardдёӢжүӢпјҢи®©йў„жөӢQеҖје’Ңзңҹе®һQеҖјзҡ„жҜ”иҫғй—®йўҳиҪ¬жҚўжҲҗи®©жЁЎеһӢе®һиҙЁдёҠеңЁжӢҹеҗҲrewardзҡ„й—®йўҳгҖӮ

DQN д»Јз Ғе®һи·ө

йҰ–е…Ҳе®ҡд№үз»ҸйӘҢеӣһж”ҫжұ зҡ„зұ»пјҢдё»иҰҒеҢ…жӢ¬еҠ е…Ҙж•°жҚ®гҖҒйҮҮж ·ж•°жҚ®дёӨеӨ§еҮҪж•°гҖӮ

з„¶еҗҺе®ҡд№үдёҖдёӘеҸӘжңүдёҖеұӮйҡҗи—ҸеұӮзҡ„ Q зҪ‘з»ңгҖӮ
