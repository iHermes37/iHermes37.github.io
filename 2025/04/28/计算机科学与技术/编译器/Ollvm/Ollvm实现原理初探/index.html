Ollvm实现原理初探

OLLVM（Obfuscator-LLVM）的核心代码混淆 Pass 主要位于以下目录中，具体路径取决于你使用的分支版本（如 llvm-4.0、llvm-5.0 等）。以下是关键目录和文件说明：

路径结构（以 llvm-4.0 分支为例）

2. 主要混淆 Pass 说明

Pass 名称 对应文件 功能
指令替换 Substitution.cpp 将简单指令（如加减）替换为复杂等效形式
虚假控制流 BogusControlFlow.cpp 插入永假条件分支和不可达代码块
控制流平坦化 Flattening.cpp 将函数控制流转为单一 switch 结构
字符串加密 StringObfuscation.cpp 加密字符串常量（部分分支提供）
算术混淆 ArithmeticObfuscation.cpp 将算术运算转换为混淆表达式

5. 其他分支的路径差异

  • LLVM 5.0+ 分支：路径可能变为 llvm/lib/Transforms/Obfuscation/（代码结构随 LLVM 升级变化）。
  • 商业版（如 Hikari）：可能新增 Pass（如间接跳转混淆）。
