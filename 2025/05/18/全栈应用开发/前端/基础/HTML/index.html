HTML

HTML 标签大全

为初学者准备的：HTML 速成_哔哩哔哩_bilibili

HTML（HyperText Markup Language）是构建网页的基础语言，提供了丰富的标签来定义网页的结构和内容。以下是 HTML5 标准 下的所有标签（按功能分类）的 详细表格，包括 标签名称、语义、用法示例、兼容性 等信息。

1. 文档结构标签

标签名 语义 用法示例 兼容性
<html> HTML 文档根元素 <html lang="en"></html> 所有浏览器
<head> 文档头部（元数据） <head><title>Page</title></head> 所有浏览器
<title> 文档标题（显示在浏览器标签页） <title>My Page</title> 所有浏览器
<body> 文档主体内容 <body><h1>Hello</h1></body> 所有浏览器
<base> 设置相对 URL 的基准路径 <base href="/assets/"> 所有浏览器
<link> 引入外部资源（CSS、favicon） <link rel="stylesheet" href="style.css"> 所有浏览器
<meta> 定义元数据（字符集、视口等） <meta charset="UTF-8"> 所有浏览器
<style> 内联 CSS 样式 <style>body { color: red; }</style> 所有浏览器
<script> 嵌入或引入 JavaScript <script src="app.js"></script> 所有浏览器
<noscript> 当 JavaScript 不可用时显示内容 <noscript>JS is disabled</noscript> 所有浏览器

总结

  • 基础标签：<div>, <span>, <p>, <a>, <img> 等是 HTML 的核心。
  • 表单与输入：<form>, <input>, <textarea>, <button> 用于数据收集。
  • 多媒体：<audio>, <video>, <canvas>, <svg> 用于富媒体展示。
  • 语义化标签：<header>, <footer>, <section>, <article> 提升可访问性和 SEO。
  • 布局：<table>, <ul>, <ol>, <dl> 用于结构化数据展示。
