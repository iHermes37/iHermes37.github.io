软提示式微调

软提示式** （修改模型输入） 不修改模型内部结构，而是在输入层（或中间层）添加一系列可训练的“虚拟token”嵌入。 Prompt Tuning 在输入序列前添加可训练向量，仅训练这些向量，方法简单，与模型解耦。
P-Tuning v1/v2 Prompt Tuning的增强版，通过一个小型LSTM/MLP（Prompt Encoder） 来生成更复杂的提示，效果更稳定。

Prompt Tuning（提示微调）

Prompt Tuning 是一种参数高效微调方法，它不修改模型内部权重，而是在输入嵌入层添加可训练的软提示（Soft Prompt）向量。这些向量作为额外的”虚拟token”，引导模型生成期望的输出。

技术原理

其中 [P1, P2, ..., Pk] 是k个可训练的连续向量（软提示），它们被拼接在输入序列之前。

Instruction Tuning（指令微调）

Instruction Tuning 是一种全参数微调或参数高效微调方法，通过在多任务、多领域的指令-响应对上微调模型，使其能够理解和遵循各种人类指令。

方面 Prompt Tuning Instruction Tuning
修改范围 仅输入提示向量 模型全部或部分参数
数据需求 单任务或少任务 大规模多任务指令数据
泛化能力 任务特定 跨任务泛化能力强
训练成本 极低 中等至高
主要目标 适应特定任务格式 学会遵循指令的能力
